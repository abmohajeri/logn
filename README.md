# Logn

**Visual intuition for algorithm complexity**

An interactive web application that helps developers understand algorithm time complexity through visualizations and a comprehensive Big-O cheat sheet.

## Features

- üìä **Big-O Cheat Sheet** - Browse 30+ algorithms organized by category:
  - Search Algorithms (Linear, Binary, Jump, Interpolation)
  - Sorting Algorithms (Bubble, Merge, Quick, Heap, and more)
  - Graph Algorithms (DFS, BFS, Dijkstra, Kruskal, Prim)
  - Tree Algorithms (BST, Traversal, Trie)
  - Data Structures (Hash Tables, Binary Heaps)
  - String Algorithms (KMP, Rabin-Karp)
  - Dynamic Programming (LCS, Edit Distance)

- üìà **Interactive Complexity Visualizer** - Compare time complexities (O(1), O(log n), O(n), O(n log n), O(n¬≤)) with interactive charts

- üìù **Detailed Algorithm Info** - Each algorithm includes:
  - Time and space complexity
  - Detailed descriptions
  - Use cases
  - Pseudo code examples

## Technologies

- **HTML5** - Structure
- **Tailwind CSS** - Styling (via CDN)
- **Chart.js** - Interactive visualizations (via CDN)
- **Vanilla JavaScript** - No frameworks, just pure JS
